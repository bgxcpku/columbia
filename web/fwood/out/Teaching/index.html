Current Courses

W4240-W6240 Data Mining Data mining is the process of searching for and using patterns in data. Given large amounts of data, automated statistical analysis tools are needed. This course is a project-based, computationally intensive course that will cover both computational methods for gathering and processing large amounts of data (briefly, prior programming experience required) and the theory and practice of developing and adapting machine learning and statistical models and techniques to datasets of interest to you. The 4000 level course is a masters level course. The 6000 level course is for PhD students and will entail extra work throughout the semester.
