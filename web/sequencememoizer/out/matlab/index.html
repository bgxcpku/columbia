Using the Sequence Memoizer with MATLAB

You can use the jar file through MATLAB. To do this you will need to first open MATLAB. If you type

version -java

into the command line it will display the JVM version MATLAB is using. If the JVM is 1.4 or lower you’ll need to upgrade to at least 1.5 in order to use this code. If the JVM version is 1.5 you should download the jar file compiled with version 1.5 otherwise download the jar file compiled with version 1.6. In the MATLAB command line type

edit classpath.txt

In the text editor, add to the bottom the path to the downloaded sequence memoizer jar. Close and re-open MATLAB. To now use the java classes defined in the jar you need to enter into the command line

import edu.columbia.stat.wood.pub.sequencememoizer.*

Now, all of the classes in the package are avaialable and can be used. Instructions on how to use java classes in the MATLAB environment can be found here.
