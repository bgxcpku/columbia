Notes

*The two versions authored by Nicholas Bartlett are compiled with different versions of the JDK. The jars contain the gnu.trove package as well as the edu.columbia.stat.wood.pub.sequencememoizer package. There are no example applications included in the jar, so you must implement your own application in MATLAB, R, or Java that uses this code. Example usage is given on the Matlab page. This example is easily translated into Java usage. *The C++ version’s documentation is found inside the tarball. *The C++ version includes Python bindings.
