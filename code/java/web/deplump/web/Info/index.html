The deplump family of compressors is currently under development in a number of academic institutions and laboratories around the world.

Please direct all enquiries (commercial and academic) to fwood@stat.columbia.edu.

For background information on how deplump was developed and the technology used therein, refer to the following papers

  • Forgetting counts : Constant Memory inference for a dependent hierarchical Pitman-Yor Process . N. Bartlett, D. Pfau, and F. Wood, ICML, 2010 PDF
  • Lossless compression based on the Sequence Memoizer. J. Gasthaus, F. Wood, and Y.W. Teh. DCC, 2010. PDF
  • A Stochastic Memoizer for Sequence Data. F. Wood, C. Archambeau, J. Gasthaus, L. James, and Y.W. Teh. ICML, 2009. PDF, Talk, Video Lecture
