Important Disclaimers

  • Any data you send us using this form will be sent back and forth to our server in the clear and is therefore vulnerable to intercept. Don’t deplump sensitive data.
  • Absolutely do not discard your original data. Deplump is alpha code undergoing constant development. There is a very high chance that tomorrow’s version of deplump will be incompatible with today’s. This means you might not ever be able to recover your original data if you were silly enough to discard it after this warning. That would be miserable for all of us, so treat this as it is; an easy way to try out an awesome new lossless compression technology.
  • That said, we save neither the file you upload nor its filename. We do save its size and mimetype. We also save the ip address from which the upload originated.
